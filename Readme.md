## 📦 Installation

1. **Clone the repo**
```bash
git clone https://github.com/your-username/ecommerce-app.git
cd ecommerce-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**
Create a `.env` file based on `.env.example` and add your keys (Stripe, DB URI, etc.)

4. **Run the app**
```bash
npm run dev
```
